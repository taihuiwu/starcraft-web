## 五、数值校验备注

### 5.1 与SC1原版的对比差异

| 项目 | SC1原版 | 本项目 | 差异说明 |
|------|---------|--------|---------|
| 游戏tick率 | 24fps | 24fps | 一致 |
| 初始水晶 | 50 | 50 | 一致 |
| 最大人口 | 200 | 200 | 一致 |
| Marine HP | 40 | 40 | 一致 |
| Marine 攻速 | 15tick | 15tick | 一致 |
| Zergling 每次2只 | 是 | 是 | 一致 |
| 房价(Overlord) | 100矿 | 100矿 | 一致(提供8人口) |
| 龙骑射程 | 5 | 5 | SC1原版需升级(+1→6) |
| 航母拦截机 | 4/8(升级后) | 4/8(升级后) | 一致 |
| 金甲虫伤害 | 100 | 100 | 一致(升级后125) |

### 5.2 关键平衡参数

- **攻速单位**: tick数（数值越小=攻击越快）。例如Marine的15tick意味着每秒攻击1.6次
- **伤害公式**: 最终伤害 = 基础攻击力 × 攻击类型系数 - 护甲值（最小伤害1点）
- **护盾恢复**: 脱离战斗7秒后开始恢复，恢复速率0.015/tick（建筑）
- **能量恢复**: 0.5625/tick（所有能量单位统一）
